# ai-stuff

A public collection of **articles, links, experiments, and notes about AI**.

This is my open notebook on AI Engineering: what I read, what I try, what breaks,
and what I learn on the path from Fullstack to AI Engineer. Loose and incremental
content — not a product, just a place to think in public.

> Maintained by [Thiago Marinho](https://github.com/tgmarinho).

---

## What lives here

| Folder | Content |
|---|---|
| `articles/` | Longer-form articles and notes |
| `links/` | Curated, commented links (papers, posts, videos, tools) |
| `experiments/` | Small, self-contained code experiments |
| `notes/` | Quick notes, summaries, takeaways |

Folders are created as content shows up — nothing needs to exist upfront.

---

## Golden rule (public by default)

This repository is **public**. Before any commit, remember that **nothing
personal, sensitive, or privately owned by third parties goes in here**:

- ❌ No credentials: `.env`, API keys, tokens, passwords, internal URLs.
- ❌ No personal data (mine, customers', or third parties').
- ❌ No paid-course material, private repos, or third-party proprietary content.
  Link the source ✔️, copy the content ❌.
- ✔️ Ideas, learnings, my own code, public links, and properly credited references.

When using third-party material, **give credit and link the source** — respecting
its original license.

---

## How to contribute (myself / agents)

- Content is written in **English** (this repo's language).
- Prefer Markdown.
- Each code experiment is self-contained in its own subfolder with a short
  `README.md` (what it is, how to run it).
- Detailed rules for AI agents in [`AGENTS.md`](./AGENTS.md).
